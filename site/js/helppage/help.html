Instruction to Maptaxonomy:

1. Insert web-address of your google spreadsheet in the first window and page name + range in the second

Remember to insert page name and range only in this format: Pagename!A1:С

2. Press "Send" button

3. Now you can get the basic information by clicking on the marker.
In the extra window you can choose some parametres for filtration and click "Apply" button.

Remember to insert date only in formats: "yyyy-mm-ddThh:mm:ssZ" or "Fri Jun 22 2018 15:54:16 GMT+0300 (RTZ 2 (зима))"

4. Clicking on the "Clear" button, you delete everything concerning your data.

Instruction to table structure:

1. In your spreadsheet columns name must contain key words: 'genbank', 'name', 'position', 'latitude', 'longitude', 'date','int','float' and 'str'. If there are no column 'latitude' or 'longitude', nothing will be painted. If column name doesn't contain key word, it will be treated as a string. If there is more than one column 'latitude' or 'longitude', only the last one will be considered.
